Key Features

Commit-Based Task Tracking

Automatically create and update tasks based on your commit messages. Simply follow a specific format and AutoTrack will do the rest.

Kanban Board

Visualize your project progress with a Kanban board that automatically updates as tasks move through different stages.

Team Collaboration

Create teams, assign tasks, and track progress together. Perfect for collaborative projects.

Tagging and Milestones

Organize tasks with tags and group them into milestones for better project management.

How It Works

  1. Connect Your GitHub Repository - Link your GitHub repo to AutoTrack through OAuth.
  2. Format Your Commit Messages - Use the special format: 
    Feature01 : Create login page -> Mahi -> todo
  3. AutoTrack Does the Rest - Tasks are automatically created and updated based on your commits.
  4. Visualize and Manage - Use the Kanban board to get a clear overview of your project's progress.
